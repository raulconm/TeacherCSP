Conducir de Málaga a Santander
==============================

..	index::
	single: CodeLens
	
Imagínate que estás planeando un viaje de Málaga a Santander. Si sabes cuántos kilómetros por litro puede recorrer tu coche, y cuántos kilómetros de distancia hay de una ciudad a otra, puedes calcular el número de litros que vas a consumir.

Utiliza el siguiente *CodeLens* para observar cómo ejecutaría un ordenador el siguiente programa:

- Pulsa |codelensfwd| para ejecutar el programa línea por línea.
- Pulsa |codelenslast| para ejecutar todas las líneas del programa de una vez.

.. codelens:: Malaga_2_Santander

   distancia = 948.1
   kpl = 16
   litros = distancia / kpl
   
..	index::
	single: camel case

Además, si sabemos cuál es el coste medio de un litro de combustible, podemos calcular cuánto nos va a costar el viaje en coche de Málaga a Santander.  

.. Note::

   Observa que estamos utilizando nombres de variables como ``costePorLitro`` y ``costeDelViaje``. Un nombre de variable no puede contener espacios, así que una forma de hacer que la variable sea más fácil de leer cuando contiene más de una palabra es escribiendo en mayúscula la primera letra de cada nueva palabra, como en ``costePorLitro`` y ``costeDelViaje``. Fíjate que esta diferencia entre mayúsculas y minúsuclas es importante en Python, porque ``costedelviaje`` y ``costeDelViaje`` serán variables distintas.   

.. codelens:: Malaga_2_Santander_Coste

   distancia = 948.1
   kpl = 16
   litros = distancia / kpl
   costePorLitro = 1.22
   costDelViaje = litros * costePorLitro
   
..	index::
	single: tracing
	single: string
	single: print
	pair: programming; tracing
	
Esto que hemos hecho se llama **trazar** el programa. Normalmente lo que hacemos es **ejecutar** un programa -- le decimos al ordenador que ejecute cada paso del programa tan rápido como pueda. Cuando lo hacemos, no podemos ver los valores que va tomando cada variable, como hemos hecho aquí. Podemos saber qué valores tiene cada variable haciendo que se muestren en la pantalla del ordenador. La función ``print`` puede tomar una *entrada* (el nombre de una variable indicada entre paréntesis) y mostrar su contenido, su valor, por pantalla. Con ``print`` también se puede imprimir en pantalla una **cadena de caracteres** (como ``"Coste para ir de Málaga a Santander"``), que es una secuencia de caracteres delimitada por comillas, como en la línea 6. Esto imprimirá el contenido exacto de la cadena, y es muy útil para indicar qué valores se están mostrando.
