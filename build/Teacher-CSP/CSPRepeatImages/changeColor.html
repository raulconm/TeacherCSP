csp-11-4-1: Another way to get a similar effect to increasing the red, is to decrease the green and blue.  Figure out how to do that in the program above and then use that information to drag the code blocks below from the left to the right in the correct order with the correct indention.


        

        
from image import *
img = Image("beach.jpg")
pixels = img.getPixels()\nfor p in pixels:
    g = p.getGreen()\nb = p.getBlue()
    p.setGreen(g * 0.75)\np.setBlue(b * 0.75)
    img.updatePixel(p)
win = ImageWin(img.getWidth(),img.getHeight())\nimg.draw(win)
