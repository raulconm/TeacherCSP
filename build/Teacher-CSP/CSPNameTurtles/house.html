Pedro construye una casa¶

Vamos a combinar a hora el código del cuadrado y el del triángulo para crear un dibujo sencillo de una casa.

Note

El siguiente programa tiene líneas en blanco alternadas con líneas de código. El ordenador ignora estas líneas en blanco. Las añadimos en los programas para agrupar las partes del código que hacen algo concreto, como dibujar un cuadrado, en este caso. Normalmente estas agrupaciones van precedidas de un comentario (comienzan con el caracter #), para indicar qué hace esa parte del código.





  



 (Turtle_House) 








Imagina que quieres crear otro cuadrado incompleto en el tejado para dibujar una “chimenea”.



        

        
csp-5-2-1: Coloca los siguientes bloques de código paa dibujar una casa y su chimenea, como en la imagen de arriba. Dibuja primero un cuadrado para el cuerpo de la casa, a continuación dibuja el tejado, y finalmente dibuja la chimenea.



        

        
from turtle import *\nespacio = Screen()\npedro = Turtle()
# Dibuja un cuadrado\npedro.forward(100)\npedro.right(90)\npedro.forward(100)\npedro.right(90)\npedro.forward(100)\npedro.right(90)\npedro.forward(100)
# Coloca para dibujar el tejado\npedro.right(90)
# dibuja el tejado\npedro.forward(100)\npedro.right(-120)\npedro.forward(100)\npedro.right(-120)\npedro.forward(100)\npedro.right(-120)
# Coloca para dibujar la chimenea\npedro.right(-60)\npedro.forward(40)\npedro.setheading(90)
# Dibuja la chimenea\npedro.pencolor("red")\npedro.forward(30)\npedro.right(90)\npedro.forward(30)\npedro.right(90)\npedro.forward(30)\npedro.right(90)
